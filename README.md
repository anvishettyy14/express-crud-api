# Express CRUD API

A simple CRUD API built using Node.js and Express.js.

## Features

- Get all posts
- Get post by ID
- Create new post
- Update post
- Delete post

## Technologies Used

- Node.js
- Express.js

## API Endpoints

GET /posts

GET /posts/:id

POST /posts

PUT /posts/:id

DELETE /posts/:id
